Pleiades Places API

Chunks of read-only data we use in various applications are available by HTTP in the JSON format.

Status

A count of places, locations, and names

http://pleiades.stoa.org/api/status

Access Example

$ curl -v http://pleiades.stoa.org/api/status
> GET /api/status HTTP/1.1
> Host: pleiades.stoa.org
> Accept: */*
> 
< HTTP/1.1 200 OK
< Date: Fri, 20 Jan 2012 21:07:51 GMT
< Content-Type: application/json
< Cache-Control: max-age=3600
< Access-Control-Allow-Origin: *
< 
{"num_places": 34082, "num_locations": 35213, "num_names": 26646}

Data Items

num_places
The current number of published places (int)
num_locations
The current number of published locations (int)
num_names
The current number of published names (int)
